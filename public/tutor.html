Tutorial

Ikuti langkah demi langkah untuk menghubungkan akun Telegram Anda ke sistem tracking.

1

Dapatkan ID Telegram Anda

Banyak pengguna bingung di sini, ikuti cara mudah ini:

  1. Buka Telegram, cari bot bernama @userinfobot
  2. Klik tombol START
  3. Bot akan membalas dengan beberapa baris data. Perhatikan angka di sebelah tulisan Id:
  4. Salin (copy) angka tersebut. Itulah ID Telegram unik Anda.
2

Aktifkan @trackhayabot

Sistem hanya bisa mengirim pesan jika Anda sudah menyapa bot utama:

Cari & Jalankan

Cari @trackhayabot lalu klik START.

Fungsi

Bot ini berfungsi sebagai pengingat (alarm) dan tempat menginput data permainan

3

Input Data di Website

Kembali ke halaman dashboard website Anda dan isi:

ID Telegram (Angka yang Anda copy dari Step 1)
Menit Notifikasi (Berapa menit sekali alarm dikirim)

Isi juga Saldo Awal, Stop-Win, dan Stop-Loss.

Lalu klik MULAI TRACKING.
